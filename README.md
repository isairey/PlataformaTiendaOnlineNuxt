# 🌌 Acerca del proyecto

**Nuxt Store** es una tienda en línea desarrollada con Nuxt.js que permite explorar productos, navegar de manera dinámica y disfrutar de una experiencia web rápida y moderna.

La aplicación está diseñada utilizando tecnologías modernas del ecosistema Vue y Nuxt para brindar:

- 🛍️ Catálogo de productos
- ⚡ Navegación rápida
- 📱 Diseño responsive
- 🧭 Routing dinámico
- 🎨 Interfaz moderna
- 🚀 Renderizado optimizado
- 🔥 Arquitectura escalable

```bash
NuxtStore/
│
├── assets/                 # Recursos estáticos
├── components/             # Componentes reutilizables

├── layouts/                # Layouts de la aplicación
├── pages/                  # Páginas y rutas
├── public/                 # Archivos públicos
├── server/                 # Backend y API
```

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/nuxt-store.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd nuxt-store
```

---

## 3️⃣ Instalar dependencias

### NPM

```bash
npm install
```

### PNPM

```bash
pnpm install
```

### Yarn

```bash
yarn install
```

### Bun

```bash
bun install
```

---

# ▶️ Ejecutar servidor de desarrollo

### NPM

```bash
npm run dev
```

### PNPM

```bash
pnpm dev
```

### Yarn

```bash
yarn dev
```

### Bun

```bash
bun run dev
```

Abrir en:

```bash
http://localhost:3000
```
